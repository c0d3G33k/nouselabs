# How to use? 

- Save `lab.com` as a local entry in hosts file
- Run git clone
- Navigate to cloned folder
- Run `docker build . -t lab7`
- Run `docker run -d -p 80:80 lab7`
- Open `http://lab.com/`

# Other way? 

- Install nodejs & npm
- Run git clone
- Navigate to cloned folder
- Run `npm install`
- Run `node index.js`
- Open `http://lab.com/`
