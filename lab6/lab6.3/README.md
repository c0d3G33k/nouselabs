# How to use? 

- Save `lab.com` as a local entry in hosts file
- Run git clone
- Navigate to cloned folder
- Run `docker build . -t lab6.3`
- Run `docker run -d -p 8000:8000 lab6.3`
- Open `http://lab.com:8000/`

# Other way? 

- Install nodejs & npm
- Run git clone
- Navigate to cloned folder
- Run `npm install`
- Run `node index.js`
- Open `http://lab.com:8000/`
